Here are some data visualizations of the Ashley Madison website data dump.

Sepy 11 , 2015

The recent development regarding the Ashley Madison website presents an interesting opportunity for me to practice my newly learned skills in data visualization. I downloaded the data dumps that "The Impact Team" has put out to the torrent site. After many hours of importing all the data dumps, I am able to query the database to create some of these data visualizations shown here.

The data needs a lot of cleansing, however. The account profiles do not seem to contain correct information. For example, the geographical information such longitude and latitude do not all match up with the member's address information.

As I complete the SQL queries to pull the data that I'm interested in, I create specific visualization to show the information I want to present. Each of those tidbit of information is shown as links on the nav bar on the left.
